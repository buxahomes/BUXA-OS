# Purpose

This document defines how Artificial Intelligence retrieves and applies official PDMS knowledge within the BUXA Business Operating System.

PDMS prompts do not replace project delivery.

They retrieve structured PDMS knowledge and assist project teams in applying official standards consistently throughout the project lifecycle.

The purpose of PDMS prompts is to improve project consistency, reduce uncertainty and support high-quality project delivery.

---

# Vision

Artificial Intelligence should become an intelligent assistant for the BUXA PDMS system.

Every recommendation should originate from official PDMS standards.

Every project should benefit from the same structured delivery methodology.

Artificial Intelligence should support project delivery without replacing professional judgement.

---

# PDMS Philosophy

Projects should follow systems rather than individual experience.

PDMS defines the standards.

PDMS prompts retrieve those standards.

Artificial Intelligence applies them consistently.

Project evidence continuously improves the PDMS system.

# Project Context Assembly

Project guidance should always be generated from complete project context rather than isolated instructions.

Project context may include:

- Project Stage
- Inspection Records
- Measurements
- Photos and Evidence
- Product Information
- Designer Requirements
- Client Requirements
- Previous Decisions
- Quality Standards

Artificial Intelligence should understand the project before making recommendations.

Better project context leads to better project guidance.

Context should always originate from official PDMS records.

---

# PDMS Validation

Before providing project guidance, Artificial Intelligence should confirm that:

- Official PDMS procedures are being used.
- Relevant project stage is identified.
- Recommendations align with project standards.
- Required evidence is considered.
- Related inspection records are referenced where appropriate.

Project guidance should remain consistent with official PDMS standards.

# PDMS Principles

Every PDMS prompt should:

- Retrieve official PDMS knowledge.
