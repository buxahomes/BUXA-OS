# Purpose

This document defines the official Inspection Standards used during **Stage 01 — Project Initiation** within the BUXA PDMS™ Project Delivery Management System.

These standards define how each Control Point shall be verified before a project is approved to proceed to Stage 02.

Inspection Standards establish consistent verification requirements across all projects.

# IS-001

## Inspection Item

项目基本信息确认

---

## Related Control Point

CP-001

---

## Inspection Objective

确认项目基础信息完整、准确，并满足项目启动要求。

---

## Inspection Method

Document Review

---

## Inspection Tools

- Project Information Form
- Client Information Form

---

## Inspection Location

Project Documentation

---

## Inspection Frequency

Once before project initiation.

---

## Acceptance Criteria

The following information shall be complete:

- Project name
- Project address
- Client information
- Project type
- Responsible FPE

---

## Required Evidence

Completed Project Information Form.

---

## Non-Conformance Action

Project initiation shall not proceed until all required information has been completed.

---

## Notes

None.

# Stage Summary

Stage 01 Inspection Standards establish the verification requirements for project initiation.

The standards ensure that:

- Project information is complete.
- Project participants have been identified.
- Client requirements are documented.
- Initial project risks have been reviewed.
- The project is ready to enter Stage 02.

Successful completion of all applicable Inspection Standards enables Quality Gate 01 approval.
