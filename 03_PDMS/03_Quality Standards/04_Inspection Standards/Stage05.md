# Purpose

This document defines the official Inspection Standards used during **Stage 05 — Installation Delivery** within the BUXA PDMS™ Project Delivery Management System.

These standards establish the inspection requirements used to verify that flooring installation is performed in accordance with approved drawings, technical requirements and BUXA PDMS™ Quality Standards.

The objective is to ensure consistent workmanship, controlled execution and high-quality project delivery throughout the installation process.

## Inspection Objective

确认安装工作按照批准方案正式开始。

---

## Inspection Method

## Inspection Tools

# Stage Summary

Stage 05 Inspection Standards establish the process quality requirements for flooring installation.

These standards verify:

- Installation commencement
- Installation direction
- Expansion gaps
- Installation methods
- Daily workmanship
- Site coordination
- Installation progress
- Finished surface protection
- Installation completion

Successful completion of all applicable Inspection Standards enables Quality Gate 05 approval.
