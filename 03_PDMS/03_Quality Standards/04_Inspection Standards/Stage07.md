# Purpose

This document defines the official Inspection Standards used during **Stage 07 — Project Handover** within the BUXA PDMS™ Project Delivery Management System.

These standards establish the requirements for the formal handover of a completed flooring project.

Project handover confirms that the homeowner has received the completed flooring system, supporting documentation and maintenance guidance in accordance with the BUXA PDMS™ delivery process.

Verify that the completed flooring project has been formally handed over to the homeowner and that all project documentation, maintenance guidance and warranty information have been successfully delivered.

# Stage Summary

Stage 07 Inspection Standards establish the formal project handover requirements.

These standards verify:

- Project readiness
- Customer walkthrough
- Documentation handover
- Maintenance guidance
- Warranty briefing
- Customer acceptance
- Project records
- Aftercare activation

Successful completion of all applicable Inspection Standards enables Quality Gate 07 approval.
