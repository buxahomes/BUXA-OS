# Purpose

This directory defines the official Quality Management System (QMS) used within BUXA PDMS™.

It establishes how quality is planned, controlled, inspected, documented and continuously improved throughout every high-end residential flooring project.

Rather than relying on individual judgement, every project follows one consistent quality management methodology.

---

# Vision

Outstanding projects are achieved through outstanding quality management.

Quality is not created during the final inspection.

Quality is created throughout the entire project lifecycle.

BUXA PDMS™ treats quality as a continuous process rather than a single event.

Every stage contributes to the final project outcome.

---

# Quality Philosophy

BUXA PDMS™ believes that quality should be:

Planned.

Measured.

Verified.

Documented.

Continuously Improved.

Quality is never assumed.

Quality is always demonstrated through evidence.

---

# Objectives

The Quality Management System exists to:

- Prevent defects before they occur.
- Standardise inspection procedures.
- Reduce project risk.
- Improve communication.
- Support objective decision making.
- Create consistent project outcomes.
- Protect long-term customer satisfaction.

# Quality Architecture

The Quality Management System consists of five integrated components.

```
Inspection Framework

↓

Quality Gates

↓

Control Point Library

↓

Inspection Standards

↓

Inspection Reports
```

Each component performs a unique role while supporting one unified quality system.

---

# System Components

## 01 Inspection Framework

Defines the overall inspection methodology.

Explains when inspections occur, who performs them and why they are required.

---

## 02 Quality Gates

Defines the approval checkpoints between project stages.

Projects only proceed after the relevant Quality Gate has been successfully completed.

---

## 03 Control Point Library

Defines every inspection checkpoint used throughout project delivery.

Each Control Point represents one measurable requirement.

The library is designed for continuous expansion.

---

## 04 Inspection Standards

Defines the acceptance criteria for every Control Point.

Inspection Standards explain how quality is evaluated objectively.

---

## 05 Inspection Reports

Defines how inspection results are documented.

Reports provide evidence that Quality Gates have been satisfied.
