# Purpose

This document defines all Control Points used during **Stage 01 — Project Initiation** within the BUXA PDMS™ Project Lifecycle.

Project Initiation establishes the project foundation.

Every Control Point in this stage verifies that the project can safely proceed to the Design Review stage.

# Stage Objective

The objective of Stage 01 is to establish a complete and accurate understanding of the project before any technical review or material planning begins.

Project scope, stakeholders, customer expectations and basic project information should all be confirmed during this stage.

Stage 01 contains ten Control Points.

These Control Points establish the project foundation and provide the evidence required for Quality Gate 01.

Only after all required Control Points have been successfully verified may the project proceed to Stage 02 — Design Review.
