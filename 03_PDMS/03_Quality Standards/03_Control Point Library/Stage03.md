# Purpose

This document defines all Control Points used during **Stage 03 — Material Planning** within the BUXA PDMS™ Project Lifecycle.

The purpose of Material Planning is to ensure that all materials required for the project have been accurately confirmed, planned and coordinated before installation preparation begins.

This stage focuses on delivery readiness rather than purchasing activities.

# Stage Objective

The objective of Stage 03 is to ensure that every material, accessory and delivery requirement has been confirmed before installation begins.

No installation should proceed without a complete and coordinated material plan.

# Stage Summary

Stage 03 ensures that every material required for project delivery has been correctly planned before installation preparation begins.

The Control Points within this stage verify:

- Product selection
- Product specifications
- Material quantities
- Waste allowance
- Accessories
- Delivery planning
- Site receiving conditions
- Material storage
- Installation resources

Successful completion of these Control Points enables Quality Gate 03 approval.

Only approved projects proceed to Stage 04 — Installation Preparation.
