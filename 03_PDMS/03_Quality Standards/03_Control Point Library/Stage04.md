# Purpose

This document defines all Control Points used during **Stage 04 — Installation Preparation** within the BUXA PDMS™ Project Lifecycle.

The purpose of Installation Preparation is to verify that the project site is fully prepared before flooring installation begins.

No installation should commence until all preparation requirements have been successfully verified.

# Stage Objective

The objective of Stage 04 is to eliminate installation risks before work begins.

This stage confirms that the project environment, subfloor, materials and site conditions satisfy the requirements for professional flooring installation.

# Stage Summary

Stage 04 verifies that the project is fully prepared before installation begins.

The Control Points within this stage focus on:

- Site readiness
- Subfloor condition
- Environmental conditions
- Material verification
- Material storage
- Installation equipment
- Installation planning

Successful completion of these Control Points enables Quality Gate 04 approval.

Only approved projects proceed to Stage 05 — Installation.
