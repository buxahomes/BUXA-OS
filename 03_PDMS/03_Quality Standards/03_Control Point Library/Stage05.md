# Purpose

This document defines all Control Points used during **Stage 05 — Installation Delivery** within the BUXA PDMS™ Project Lifecycle.

The purpose of Installation Delivery is to ensure that flooring installation is carried out in accordance with the approved design, technical requirements and BUXA PDMS™ Quality Standards.

This stage focuses on delivery management, workmanship quality and process control throughout the installation.

# Stage Objective

The objective of Stage 05 is to ensure that flooring installation is executed safely, accurately and consistently.

Installation quality is achieved through structured project coordination, continuous inspection and proactive issue management.

# Stage Summary

Stage 05 manages the complete flooring installation process.

The Control Points within this stage verify:

- Installation commencement
- Installation direction
- Expansion gaps
- Installation methods
- Workmanship quality
- Site coordination
- Installation progress
- Finished surface protection
- Installation completion

Successful completion of these Control Points enables Quality Gate 05 approval.

Only approved projects proceed to Stage 06 — Post-Installation Inspection.
