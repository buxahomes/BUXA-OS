# Purpose

This document defines all Control Points used during **Stage 06 — Post-Installation Inspection** within the BUXA PDMS™ Project Lifecycle.

The purpose of Post-Installation Inspection is to verify that the completed flooring installation satisfies the approved design, technical requirements and BUXA PDMS™ Quality Standards before project handover.

Inspection at this stage determines whether the project is ready to proceed to formal delivery.

# Stage Objective

The objective of Stage 06 is to perform a comprehensive quality verification of the completed flooring system.

Any identified non-conformities shall be recorded, rectified and re-inspected before project handover.

# Stage Summary

Stage 06 verifies that the completed flooring installation satisfies all BUXA PDMS™ Quality Standards before project handover.

The Control Points within this stage verify:

- Overall installation quality
- Surface condition
- Joint quality
- Transition details
- Skirting installation
- Finished surface protection
- Defect rectification
- Inspection documentation
- Final quality verification

Successful completion of these Control Points enables Quality Gate 06 approval.

Only approved projects proceed to Stage 07 — Project Handover.
