# Purpose

This document defines all Control Points used during **Stage 08 — Aftercare** within the BUXA PDMS™ Project Lifecycle.

The purpose of Aftercare is to ensure the long-term performance of the flooring system following project handover.

Aftercare provides ongoing technical support, maintenance guidance and warranty coordination, reinforcing BUXA's commitment to long-term project quality and customer satisfaction.

# Stage Objective

The objective of Stage 08 is to maintain the long-term performance of the completed flooring system.

# Stage Summary

Stage 08 ensures that project delivery continues beyond completion through structured aftercare.

The Control Points within this stage verify:

- Customer follow-up
- Product performance
- Maintenance guidance
- Aftercare issue management
- Warranty coordination
- Customer satisfaction
- Project review
- Knowledge management
- Continuous improvement
- Project closure

Successful completion of these Control Points completes the BUXA PDMS™ Project Delivery Lifecycle.
