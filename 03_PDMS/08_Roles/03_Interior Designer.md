# Purpose

This document defines the official role of the Interior Designer within the BUXA PDMS™ Project Delivery Management System.

The Interior Designer is responsible for defining the design intent of the project.

BUXA PDMS™ supports the designer by providing a structured delivery system that transforms approved designs into high-quality project outcomes.

The Interior Designer remains the creative leader throughout the design process.

The Flooring Project Engineer ensures that the design can be successfully delivered.

---

# Vision

Outstanding interior design deserves outstanding project delivery.

The purpose of BUXA PDMS™ is not to replace designers.

Its purpose is to help designers achieve the project they envisioned.

The relationship between the Interior Designer and the Flooring Project Engineer is collaborative rather than hierarchical.

---

# Role Definition

The Interior Designer defines the project's design vision.

Responsibilities include:

- Interior design
- Material selection
- Space planning
- Detail design
- Client communication
- Design approval

The designer creates the vision.

The Flooring Project Engineer delivers the vision.

# Primary Responsibilities

## 1. Design Development

Develop the complete interior design concept.

Including:

- Spatial planning
- Material selection
- Colour schemes
- Design detailing
- Client presentations

---

## 2. Flooring Selection

Select flooring products appropriate for the overall design.

Including:

- Colour
- Texture
- Surface finish
- Pattern
- Design compatibility

---

## 3. Drawing Development

Prepare project documentation.

Including:

- Floor plans
- Elevations
- Detail drawings
- Material schedules
- Design revisions

---

## 4. Design Coordination

Work collaboratively with the Flooring Project Engineer.

Including:

- Reviewing technical recommendations
- Confirming implementation details
- Resolving design conflicts
- Approving technical adjustments where required

---

## 5. Client Communication

Guide the client through design decisions.

Including:

- Design presentations
- Material confirmation
- Design revisions
- Final design approval

# Relationship with Flooring Project Engineer

The Interior Designer and the Flooring Project Engineer perform different responsibilities.

The Designer defines the design intent.

The Flooring Project Engineer verifies technical feasibility and manages project delivery.

Both roles collaborate throughout the project.

Neither role replaces the other.

---

# Relationship with Installation Team

The Installation Team does not interpret design independently.

The Flooring Project Engineer communicates installation requirements based on the approved design.

Where design clarification is required, the Interior Designer provides direction through the Flooring Project Engineer.

---

# Relationship with Homeowner

The Interior Designer supports the homeowner throughout the design process.

The Flooring Project Engineer supports the homeowner throughout the delivery process.

Together they provide a seamless project experience.

---

The Interior Designer is an essential project participant within the BUXA PDMS™ system.

The system provides technical expertise, structured delivery and quality management, allowing designers to focus on creating exceptional spaces.
