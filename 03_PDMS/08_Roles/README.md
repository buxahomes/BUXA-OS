# Purpose

This directory defines the official project participants within the BUXA PDMS™ Project Delivery Management System.

Project delivery is achieved through structured collaboration rather than individual effort.

Each participant has clearly defined responsibilities, decision authority and communication boundaries.

Together they create one integrated project delivery system.

---

# Vision

A successful project is not the result of one person.

It is the result of multiple participants working within one unified operating system.

BUXA PDMS™ establishes clear responsibilities for every participant while ensuring that all communication, technical decisions and quality management remain coordinated throughout the project lifecycle.

---

# Project Delivery Philosophy

BUXA PDMS™ is built upon one simple principle:

Every participant performs a different role.

Every participant contributes to the same project outcome.

Responsibilities should never overlap unnecessarily.

Communication should always remain structured.

Project delivery should always remain controlled.

---

# Participant Overview

## 01 — BUXA

BUXA owns and continuously develops the Project Delivery Management System.

Responsibilities include:

- Product System
- PDMS™ Framework
- Technical Standards
- Quality Standards
- Training
- Documentation
- Continuous Improvement
- Customer Support

BUXA defines the system.

---

## 02 — Flooring Project Engineer (FPE)

The Flooring Project Engineer is the operational core of the project.

Responsibilities include:

- Design Coordination
- Technical Review
- Project Planning
- Installation Management
- Quality Verification
- Project Handover
- Aftercare Coordination

The FPE delivers the system.

---

## 03 — Interior Designer

The Interior Designer defines the design intent of the project.

Responsibilities include:

- Interior Design
- Material Selection
- Design Coordination
- Design Review
- Client Communication

The Designer creates the vision.

---

## 04 — Installation Team

The Installation Team performs flooring installation according to the BUXA PDMS™ standards.

Responsibilities include:

- Flooring Installation
- Site Coordination
- Daily Progress
- Self-Inspection
- Rectification

The Installation Team executes the work.

---

## 05 — Homeowner

The Homeowner is the final recipient of the completed project.

Responsibilities include:

- Project Confirmation
- Design Decisions
- Final Acceptance
- Aftercare Participation

The Homeowner experiences the delivery.

Every participant has clearly defined responsibilities.

Responsibilities should complement one another rather than overlap.

The goal is collaborative project delivery rather than individual project management.

# Governance

Every participant shall follow the standards, workflows and quality requirements defined by BUXA PDMS™.

Role responsibilities may evolve over time.

However, the relationship between participants should remain consistent.

Every project should have:

- One BUXA System
- One designated Flooring Project Engineer
- One coordinated Installation Team
- One Interior Designer (where applicable)
- One Homeowner
