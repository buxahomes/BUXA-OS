# Purpose

This document defines the official role of the Flooring Project Engineer (FPE) within the BUXA PDMS™ Project Delivery Management System.

The Flooring Project Engineer is the primary project delivery owner responsible for coordinating every stage of a flooring project, from design coordination through installation, inspection, project handover and aftercare.

The FPE ensures that every project is delivered according to approved design intent, technical standards and the BUXA PDMS™ Quality Standards.

---

# Vision

Outstanding flooring projects are achieved through professional project delivery rather than installation alone.

The Flooring Project Engineer bridges the gap between design and construction, ensuring that every project is delivered through a controlled, predictable and repeatable process.

The FPE transforms design intent into successful project delivery.

---

# Role Definition

The Flooring Project Engineer (FPE) is the core delivery coordinator within the BUXA PDMS™ system.

The FPE manages the complete flooring project delivery process, including:

- Design coordination
- Technical review
- Project planning
- Installation management
- Post-installation inspection
- Quality verification
- Project handover
- Aftercare coordination

The installer installs the flooring.

The FPE manages the entire delivery system.

# Primary Responsibilities

## 1. Design Coordination

Coordinate with interior designers to ensure that design intent can be successfully implemented.

Responsibilities include:

- Reviewing drawings
- Confirming flooring layouts
- Confirming installation direction
- Confirming transition details
- Confirming stair details
- Resolving technical conflicts
- Supporting design implementation

---

## 2. Technical Review

Review all technical aspects before installation begins.

Including:

- Product suitability
- Site conditions
- Subfloor conditions
- Moisture risks
- Expansion requirements
- Construction feasibility
- Technical risk assessment

---

## 3. Project Planning

Prepare the flooring delivery plan.

Responsibilities include:

- Installation scheduling
- Material planning
- Delivery coordination
- Site preparation planning
- Risk identification
- Project communication planning

---

## 4. Installation Delivery Management

Manage the flooring installation process.

Responsibilities include:

- Coordinating the installation team
- Confirming installation readiness
- Monitoring installation progress
- Resolving on-site issues
- Managing technical changes
- Ensuring compliance with BUXA PDMS™ standards

The FPE manages the installation process.

The installer performs the installation.

---

## 5. Post-Installation Inspection

Conduct comprehensive inspections after installation.

Responsibilities include:

- Verifying installation quality
- Completing Control Point inspections
- Managing Quality Gates
- Identifying non-conformities
- Coordinating rectification
- Confirming project readiness for handover

Inspection confirms that the completed work satisfies the BUXA PDMS™ Quality Standards.

---

## 6. Project Handover

Manage the formal delivery of the completed project.

Responsibilities include:

- Final project walkthrough
- Customer acceptance inspection
- Maintenance briefing
- Warranty explanation
- Delivery of project documentation
- Completion confirmation

Project delivery is only considered complete after formal customer acceptance.

---

## 7. Aftercare Coordination

Support long-term project performance after completion.

Responsibilities include:

- Technical support
- Warranty coordination
- Follow-up inspections
- Customer enquiries
- Maintenance recommendations
- Long-term customer relationship management

---

## 8. Documentation Management

Maintain complete and accurate project records.

Including:

- Inspection reports
- Project photographs
- Technical records
- Quality documentation
- Handover documentation
- Aftercare records

# Relationship with Other Roles

## Interior Designer

The designer defines the design intent.

The FPE ensures that the design can be successfully delivered.

The FPE provides technical support throughout project implementation.

---

## Installation Team

The installation team performs flooring installation.

The FPE manages the installation environment, provides technical guidance, coordinates installation activities and verifies compliance with BUXA PDMS™ standards.

---

## Material Supplier

The supplier provides approved products.

The FPE verifies product suitability, coordinates deliveries and confirms material readiness.

---

## Homeowner

The homeowner receives project updates, participates in final acceptance and receives long-term aftercare support.

The FPE serves as the primary technical contact throughout the entire project lifecycle.

---

# Authority

The Flooring Project Engineer has the authority to:

- Suspend installation when technical risks are identified.
- Reject non-compliant workmanship.
- Require corrective actions before project progression.
- Approve Quality Gates after verification.
- Recommend technical adjustments necessary for successful implementation.
- Approve project handover after all delivery requirements have been satisfied.

# Single Source of Truth

This document defines the official role of the Flooring Project Engineer (FPE) within BUXA PDMS™.

Every project delivered through BUXA PDMS™ shall assign one designated Flooring Project Engineer responsible for coordinating project delivery from initiation through aftercare.

The Flooring Project Engineer is not an installation supervisor.

The Flooring Project Engineer is the project delivery owner responsible for transforming approved design into successful project outcomes.
