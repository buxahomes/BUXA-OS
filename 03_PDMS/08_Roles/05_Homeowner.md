# Purpose

This document defines the official role of the Homeowner within the BUXA PDMS™ Project Delivery Management System.

The Homeowner is the final recipient of the completed flooring project.

Throughout the project lifecycle, the Homeowner participates in key decisions, confirms important project milestones and ultimately accepts the completed project.

The BUXA PDMS™ system exists to reduce uncertainty for the Homeowner by providing a structured, transparent and professionally managed delivery process.

---

# Vision

Every homeowner deserves confidence throughout the project journey.

Rather than requiring homeowners to manage construction, BUXA PDMS™ provides a professional delivery system that allows homeowners to make informed decisions while experienced professionals manage project execution.

---

# Role Definition

The Homeowner defines the desired project outcome and participates in key project decisions.

Responsibilities include:

- Confirming project requirements
- Approving design decisions
- Confirming product selections
- Participating in project acceptance
- Supporting long-term maintenance

The Homeowner is not responsible for managing installation, technical coordination or quality inspections.

Those responsibilities are managed through the BUXA PDMS™ delivery system.

---

# Core Mission

Participate in informed decisions and enjoy a professionally delivered flooring project.

# Primary Responsibilities

## 1. Project Confirmation

Confirm the overall project objectives.

Including:

- Project requirements
- Budget expectations
- Timeline expectations
- Design objectives

---

## 2. Design Approval

Participate in key design decisions.

Including:

- Flooring selection
- Colour confirmation
- Surface finish
- Layout approval

---

## 3. Project Communication

Maintain open communication with the Flooring Project Engineer.

Including:

- Responding to project updates
- Confirming project decisions
- Raising questions when required
- Providing timely feedback

---

## 4. Project Acceptance

Participate in the final project walkthrough.

Including:

- Reviewing completed work
- Confirming project completion
- Receiving maintenance guidance
- Receiving warranty documentation

---

## 5. Long-Term Care

Support the long-term performance of the flooring system.

Including:

- Following maintenance recommendations
- Reporting issues promptly
- Participating in aftercare inspections where required

# Relationship with Flooring Project Engineer

The Flooring Project Engineer is the Homeowner's primary project contact.

The FPE is responsible for:

- Technical coordination
- Project communication
- Delivery management
- Quality verification
- Project handover

The Homeowner should communicate project-related matters through the Flooring Project Engineer.

This ensures efficient, accurate and consistent project delivery.

---

# Relationship with Interior Designer

The Interior Designer develops the design vision.

The Homeowner reviews and approves design decisions.

The Flooring Project Engineer supports the successful implementation of the approved design.

---

# Relationship with Installation Team

The Installation Team performs flooring installation.

Technical instructions, project variations and quality-related matters should be coordinated through the Flooring Project Engineer rather than communicated directly with the Installation Team.

This ensures consistent project management and protects the approved delivery process.

---

# Rights

The Homeowner has the right to:

- Receive clear project information.
- Understand the delivery process.
- Receive regular project updates.
- Review completed work.
- Receive project documentation.
- Receive warranty support.
- Receive aftercare services in accordance with the project agreement.

---

# Professional Principles

The Homeowner is encouraged to:

- Participate in key project decisions.
- Maintain open communication.
- Respect the approved delivery process.
- Raise questions whenever clarification is required.
- Support collaborative project delivery.

# Relationship with BUXA PDMS™

The Homeowner is the final beneficiary of the BUXA PDMS™ Project Delivery Management System.

The purpose of PDMS™ is to provide homeowners with confidence throughout the project lifecycle by combining professional products, structured project management and consistent quality standards.
