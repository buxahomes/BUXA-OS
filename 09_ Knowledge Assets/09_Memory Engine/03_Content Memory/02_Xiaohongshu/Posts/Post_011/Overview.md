# Post_011 Overview

## One-Sentence Summary

深化设计的价值，不是增加图纸，而是把施工现场的不确定性提前转化为明确、统一、可执行的标准。

## Core Question

深化设计到底在解决什么？

## Core Answer

解决施工开始前尚未确定的关键问题。

## Central Thesis

深化设计不是绘图工作量的增加，而是项目风险的前置管理。

## Main Message

所有应该在办公室解决的问题，都不要留到施工现场。

## Supporting Message

现场决定越少，返工、偏差、延期和成本失控的概率越低。

## Key Takeaway

深化设计，是把经验变成标准。

## Audience

- 高端住宅室内设计师
- 设计机构创始人
- 项目经理
- 工程管理人员
- 木地板及地面系统从业者
- 高端住宅业主

## Reader Transformation

阅读前：

深化设计就是画更多、更复杂的施工图。

阅读后：

深化设计是在施工前消除不确定性，让现场按标准执行。
