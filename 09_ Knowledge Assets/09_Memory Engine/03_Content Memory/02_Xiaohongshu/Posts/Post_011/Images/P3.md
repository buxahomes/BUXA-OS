# P3

## Page Role

Method Explanation

## Headline

深化真正解决的是  
七个“不确定”

## Supporting Statement

设计的完整，不只是效果图的漂亮，

而是把所有施工现场的“不确定”，提前变为“确定”。

## Seven Questions

1. 从哪里开始铺？
2. 每块板怎么排？
3. 哪里需要收口？
4. 楼梯怎么连接？
5. 门口怎么处理？
6. 柜体如何收边？
7. 哪里需要留缝？

## Closing Statement

所有的不确定，都应该消失在施工之前。

## Visual Purpose

把抽象的“不确定性管理”转化为七个具体、可识别的施工问题。
