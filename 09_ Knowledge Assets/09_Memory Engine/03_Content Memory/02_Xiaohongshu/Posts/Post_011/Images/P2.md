# P2

## Page Role

Misconception Correction

## Headline

深化设计  
不是画更多图纸

## Supporting Copy

很多人认为：

深化设计就是把CAD画得更复杂。

其实不是。

真正的深化，

是在施工开始之前，

把所有需要现场决定的问题，

提前决定。

## Comparison

### 没有深化设计

- 施工现场临时决定
- 经验不统一，做法不一致
- 效果偏差，返工风险高
- 工期不确定，成本增加

### 有深化设计（PDMS）

- 所有关键问题提前解决
- 图纸统一，标准一致
- 效果可控，返工减少
- 工期可控，成本可控

## Closing Statement

让现场少决定，才能减少返工。

## Visual Purpose

纠正“深化设计等于增加图纸”的错误认知，并用对比建立方法价值。
