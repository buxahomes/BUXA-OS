## Node 05

### TRIM DETAIL

收边方式

提前确定不同材质、不同空间之间的收边方案，

选择合适的收边条型与安装方式，

确保过渡自然、美观耐用。

## Node 06

### SKIRTING

踢脚线

确认踢脚线的材质、造型、高度、颜色及安装方式，

与整体设计风格统一，

避免后期不匹配或返工。

## Node 07

### DOOR THRESHOLD

门口收边

明确门口收边的型材、高度差处理和安装方式，

确保开关门顺畅，

视觉整洁一致。

## Node 08

### INSTALLATION SEQUENCE

施工顺序

制定合理的施工顺序，

明确各工序的先后关系，

协调各专业配合，

避免交叉作业造成的损坏和延误。

## Bottom Quote

而是把问题留在施工前解决。
