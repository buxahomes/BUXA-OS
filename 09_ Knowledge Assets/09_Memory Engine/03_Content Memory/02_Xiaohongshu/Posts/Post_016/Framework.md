# Post_016 Framework

## Framework Name

PDMS METHOD 01

## Chinese Name

施工前八大关键节点确认法

## Framework Purpose

在地面系统正式施工前，统一设计、材料、施工和验收认知，减少现场临时决策、返工、延期与成本增加。

---

## Core Principle

好的交付，

不是施工开始以后快速解决问题。

而是在施工开始以前，

把所有关键问题提前确定。

---

# Eight Critical Confirmation Nodes

## 01 Finished Floor Level

中文：

完成面高度

### Definition

完成面高度是地板、石材、瓷砖、柜体、门、楼梯及其他界面完成后的最终标高。

### Must Confirm

- 原始结构标高
- 基层找平厚度
- 地暖结构厚度
- 胶黏剂厚度
- 地板总厚度
- 石材或瓷砖厚度
- 不同区域高差
- 门槛完成面
- 楼梯起步及收尾高度
- 柜体底部关系

### Risk if Unconfirmed

- 高低差
- 门扇无法开启
- 楼梯高度不一致
- 柜体尺寸错误
- 收口无法完成
- 现场重新找平
- 返工与增项

### Acceptance Evidence

- 标高图
- 现场测量记录
- 完成面关系图
- 责任人签字确认

---

## 02 Stair Detail

中文：

楼梯节点

### Definition

楼梯节点包括踏步、立板、楼梯鼻、压条、收口、基层与灯带等结构关系。

### Must Confirm

- 踏步宽度
- 踏步高度
- 踏步深度
- 踏步材料厚度
- 立板材料
- 楼梯鼻结构
- 收口方式
- 转角处理
- 灯带位置
- 起步与收尾关系

### Risk if Unconfirmed

- 踏步尺寸不一致
- 楼梯鼻无法安装
- 材料重新加工
- 起步高度异常
- 结构与饰面冲突
- 美观与安全风险

### Acceptance Evidence

- 楼梯节点图
- 现场复尺记录
- 样板确认
- 安装标准

---

## 03 Expansion Gap

中文：

伸缩缝预留

### Definition

为应对木地板因温度和湿度变化产生的伸缩，需要在墙边、门口及长距离铺装区域预留合理空间。

### Must Confirm

- 周边伸缩缝宽度
- 门口伸缩缝
- 大面积分区位置
- 柜体周边处理
- 墙柱周边处理
- 地暖区域要求
- 长距离连续铺装限制
- 型材覆盖能力

### Risk if Unconfirmed

- 起拱
- 变形
- 异响
- 收边外露
- 门口开裂
- 后期维修困难

### Acceptance Evidence

- 伸缩缝节点图
- 现场尺寸记录
- 安装过程照片
- 验收抽查记录

---

## 04 Material Direction

中文：

铺装方向

### Definition

根据采光、空间比例、主要视线和空间连接关系，确定地板的铺装方向与起铺逻辑。

### Must Confirm

- 主空间方向
- 自然采光方向
- 主要视觉轴线
- 走廊连接方向
- 房间间连续性
- 楼梯连接关系
- 起铺位置
- 拼花区域
- 排版逻辑

### Risk if Unconfirmed

- 空间比例失衡
- 多空间方向冲突
- 拼缝杂乱
- 材料浪费增加
- 整体视觉被破坏

### Acceptance Evidence

- 铺装方向图
- 排版图
- 起铺点标记
- 现场放线记录

---

## 05 Trim Detail

中文：

收边方式

### Definition

不同材料、不同高度和不同空间之间的连接与过渡方式。

### Must Confirm

- T型条
- Reducer
- End Cap
- 金属收口条
- 压条
- 无压条收口
- 石材交界
- 瓷砖交界
- 地毯交界
- 高低差处理

### Risk if Unconfirmed

- 收口突兀
- 材料高差
- 型材尺寸不符
- 现场临时切割
- 重复采购
- 视觉效果下降

### Acceptance Evidence

- 节点大样
- 型材样板
- 颜色确认
- 安装剖面图

---

## 06 Skirting

中文：

踢脚线

### Definition

墙面与地面的收口系统，包括明装、隐藏式及同色木皮踢脚线。

### Must Confirm

- 踢脚线款式
- 高度
- 厚度
- 材质
- 颜色
- 安装方式
- 阴阳角处理
- 门套关系
- 柜体关系
- 灯带关系

### Risk if Unconfirmed

- 与门套冲突
- 墙面收口不整齐
- 柜体缝隙
- 高度比例失衡
- 后期重新加工
- 细节不统一

### Acceptance Evidence

- 踢脚线节点图
- 材料样板
- 转角样板
- 安装验收标准

---

## 07 Door Threshold

中文：

门口收边

### Definition

门洞区域不同地面材料、标高和门套之间的衔接方式。

### Must Confirm

- 门口材料交界线
- 收边型材
- 门槛石关系
- 门扇开启高度
- 门套完成面
- 房间内外高差
- 伸缩缝位置
- 视觉连续性

### Risk if Unconfirmed

- 门扇摩擦
- 门口高低差
- 收边条位置错误
- 地板翘边
- 门套与地板缝隙
- 二次拆装

### Acceptance Evidence

- 门口节点图
- 门套与地面剖面图
- 现场尺寸复核
- 成品验收照片

---

## 08 Installation Sequence

中文：

施工顺序

### Definition

明确地面施工与墙面、木作、石材、门套、楼梯和成品保护之间的先后关系。

### Must Confirm

- 基层施工时间
- 含水率检测时间
- 地暖运行周期
- 石材完成时间
- 柜体安装时间
- 门套安装时间
- 地板进场时间
- 地板安装时间
- 踢脚线安装时间
- 成品保护时间
- 最终验收时间

### Risk if Unconfirmed

- 交叉施工
- 材料污染
- 成品损伤
- 现场等待
- 重复安装
- 工期延误
- 责任争议

### Acceptance Evidence

- 施工计划
- 工序流程图
- 交底纪要
- 节点责任表

# PDMS Pre-Construction Confirmation Process

## Step 01｜Collect Information

收集：

- 项目设计图纸
- 地面材料表
- 门表
- 柜体图
- 楼梯图
- 石材图
- 施工进度计划
- 现场测量数据

## Step 02｜Review Interfaces

审查地面系统与以下专业的接口：

- 门窗
- 石材
- 木作
- 楼梯
- 墙面
- 地暖
- 灯光
- 柜体

## Step 03｜Prepare Detailed Drawings

完成：

- 完成面标高图
- 铺装方向图
- 排版图
- 收口节点图
- 楼梯节点图
- 门口节点图
- 伸缩缝图
- 施工顺序图

## Step 04｜Organize Site Briefing

参与人员：

- 设计师
- 项目经理
- 地板项目工程师
- 施工负责人
- 木作负责人
- 石材负责人
- 门套负责人
- 业主代表

## Step 05｜Confirm Responsibility

每个节点明确：

- 谁提出方案
- 谁提供数据
- 谁确认
- 谁执行
- 谁检查
- 谁最终验收

## Step 06｜Create Confirmation Record

形成：

- 交底纪要
- 图纸版本
- 节点照片
- 尺寸记录
- 责任人
- 截止时间
- 确认状态

## Step 07｜Authorize Construction

只有当八个节点满足以下条件后，才能进入施工：

- 信息完整
- 图纸明确
- 标准统一
- 责任清晰
- 现场条件满足
- 验收方式确定

# Chain Reaction Model

现场临时决定  
→ 重新沟通  
→ 重新加工  
→ 重新采购  
→ 重新安装  
→ 重新验收  
→ 工期延长  
→ 成本增加  
→ 质量风险上升

---

# Eight-Node Formula

完成面高度  
+ 楼梯节点  
+ 伸缩缝  
+ 铺装方向  
+ 收边方式  
+ 踢脚线  
+ 门口收边  
+ 施工顺序  
= 可执行的地面交付方案
