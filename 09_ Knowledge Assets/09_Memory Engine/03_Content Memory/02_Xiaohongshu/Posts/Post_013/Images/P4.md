## Visual Description

设计师、施工、工长和供应商围绕中央“无人负责”形成环形结构。

每个角色都有不同假设，但没有唯一责任人。

## Visual Purpose

揭示现场问题的根源经常不是能力不足，而是责任边界和信息闭环缺失。
