# P2

## Page Role

Problem Introduction

## Headline

很多项目，  
不是设计出了问题。

而是施工开始以后，  
方案开始不断改变。

## Comparison Structure

### 左侧｜设计阶段 DESIGN

展示：

- 设计图纸
- 材料样板
- 已确认的方案
- 清晰的设计意图

### 右侧｜施工现场 SITE

展示：

- 未完成空间
- 临时施工状态
- 材料堆放
- 现场不确定性

## Change Arrow

CHANGE

从设计方案指向施工现场。

## Supporting Statement

一个节点调整，可能影响：

- 地板方向
- 收边方式
- 楼梯尺寸
- 柜体尺寸
- 灯带位置

## Key Conclusion

最终影响整个空间。

## Bottom Process

- 变更发生
- 影响扩散
- 连锁反应
- 整体受影响

## Visual Purpose

说明项目风险往往不是设计阶段产生，而是进入现场以后，方案不断改变并产生扩散影响。
