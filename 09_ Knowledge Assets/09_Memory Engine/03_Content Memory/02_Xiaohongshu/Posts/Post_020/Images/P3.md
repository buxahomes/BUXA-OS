# P3

## Page Type

Myth Correction / Comparison

## Main Title

不是所有缝，  
都叫施工误差。

## Body Copy

多层实木复合地板依然是真实木。

季节变化｜湿度变化｜温度变化

都会产生微小伸缩。

如果没有预留空间，

最后鼓起的，

往往就是整个地面。

## Left Panel

### Title

错误安装：顶死墙体

### Visual Label

鼓起变形

### Problems

- 木地板无伸缩空间
- 受热胀冷缩影响
- 地板相互挤压
- 产生鼓起、开裂、变形

## Right Panel

### Title

正确安装：预留伸缩缝

### Visual Label

预留 8–12 mm 伸缩缝

### Benefits

- 预留伸缩空间
- 木材自由伸缩
- 地面保持平整
- 延长地板使用寿命

## Core Message

没有缝，

不一定代表施工更好。

合理留缝，

才是长期稳定的前提。

## Visual Direction

- 左右对比
- 左侧表现地板顶墙后局部起拱
- 右侧表现均匀伸缩缝
- 错误一侧使用深灰色
- 正确一侧使用金色
- 两组节点均为同一空间和相似视角
- 重点突出结构差异而非装饰差异

## Comparison Logic

错误：

木材被限制  
→ 应力无法释放  
→ 相互挤压  
→ 起拱变形

正确：

预留活动空间  
→ 木材自然伸缩  
→ 应力释放  
→ 地面长期稳定

## Image Prompt
