# P2

## Page Type

Knowledge Introduction

## Main Title

为什么留缝？

## Body Copy

很多业主第一次看到安装现场都会问：

“为什么这里还有缝？”

其实，

真正专业的施工，

很多“缝”都是故意留下来的。

不是没做好。

而是为了让木材正常呼吸。

## Technical Label

预留伸缩缝

Expansion Gap

## Dimension

8–12 mm

## Environmental Factors

- 季节变化
- 湿度变化
- 温度变化

## Supporting Copy

木材会随着环境变化产生微小伸缩。

预留伸缩缝，

就是给它留出自然呼吸的空间。

保护地板，

也保护整个空间。

## Core Insight

伸缩缝不是施工误差，

而是材料属性决定的专业做法。

## Visual Direction
