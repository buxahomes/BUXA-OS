# P1

## Page Type

Cover

## Main Title

真正高级的地板  
都留了这条缝

## English / Brand Text

BUXA PDMS™

High-End Residential Project Delivery System

## Annotation

预留伸缩缝

Expansion Gap

## Signature

JOSEPH

## Core Message

通过一个容易被误解的施工细节建立认知冲突：

“高级地板为什么反而会留缝？”

## Visual Direction

- 高端住宅木地板完成效果
- 画面重点表现木地板与墙体之间的细缝
- 局部标注伸缩缝位置
- 暖色自然光
- 极简高端空间
- 木地板纹理清晰
- 墙面干净克制
- 不出现杂乱施工工具
- 画面需要同时具有完成效果与节点细节感

## Layout

- 主标题位于左上至中部
- “都留了这条缝”使用金色强调
- 右下或中右位置标注伸缩缝
- 品牌信息放置于左下
- JOSEPH 签名放置于底部

## Key Visual Hook

看似“不完美”的一条缝，

实际上代表专业施工与长期稳定。

## Image Prompt

高端现代住宅室内，浅色欧洲橡木地板铺设完成，白色极简墙面，阳光从落地窗照入，在墙体与木地板交接处清晰可见一条细窄、均匀、专业预留的伸缩缝，地板自然纹理真实，空间高级、安静、克制，暖灰与米色调，建筑摄影风格，细节特写与完整空间结合，正方形构图，适合高端住宅项目交付管理品牌宣传。
