# Post_020 Overview

## One-Sentence Summary

木地板墙边预留的伸缩缝不是施工缺陷，而是为木材因温度、湿度和季节变化产生的自然伸缩提供必要空间，从而降低起拱、变形和返工风险。

## Core Question

为什么木地板安装完成后，墙边仍然要保留缝隙？

## Core Answer

因为多层实木复合地板仍然包含真实木材，木材会受到温湿度及环境变化影响；合理预留伸缩缝，可以为木地板提供必要的尺寸变化空间。

## Central Thesis

真正高级的地面交付，不是看不到任何缝，而是每一条缝都被提前设计、合理预留并正确处理。

## Main Message

伸缩缝是木地板系统的重要功能节点。专业安装关注的不是简单地“有没有缝”，而是缝的位置、尺寸、遮盖方式和整体节点关系是否正确。

## Supporting Message

如果木地板安装时完全顶死墙体或固定构件，当木材发生膨胀时，应力无法释放，可能造成起拱、挤压、开裂、收口变形和大面积返工。
