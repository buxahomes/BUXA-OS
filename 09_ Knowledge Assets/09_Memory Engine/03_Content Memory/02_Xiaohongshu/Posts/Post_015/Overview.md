# Post_015 Overview

## One-Sentence Summary

高端住宅预算失控，往往不是因为材料价格本身，而是设计阶段没有提前确认关键交付节点，导致施工过程中不断出现返工、补货、等待与调整成本。

## Core Question

为什么项目最初的预算看起来合理，进入施工后却不断超支？

## Core Answer

因为报价通常只计算了可见的材料和施工内容，却没有充分识别节点冲突、标高偏差、收口调整、材料补货和工期延误等隐藏成本。

## Central Thesis

真正需要控制的，不只是材料成本，而是整个项目交付过程中的不确定性和隐藏成本。

## Main Message

预算控制不是施工阶段的财务问题，而是设计和交付管理必须提前解决的系统问题。

## Supporting Message

一个未确认的节点，可能同时带来重新加工、运输、安装、人工、延期与返工等连锁费用。

## Key Takeaway

设计越完整，交付越顺畅；关键节点确认得越早，预算越稳定。

## Reader Transformation

阅读前：

项目超预算通常是因为材料价格上涨或供应商报价过高。

阅读后：

项目超预算更常见的根源，是设计阶段未把施工交付复杂性和节点风险纳入预算管理。
