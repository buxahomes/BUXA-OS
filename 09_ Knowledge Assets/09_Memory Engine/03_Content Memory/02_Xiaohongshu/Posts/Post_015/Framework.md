# Post_015 Framework

## Framework Name

PDMS Hidden Cost Control Framework

中文名称：

PDMS 项目隐藏成本控制框架

## Framework Purpose

通过设计前置、节点确认和全过程交付管理，识别并减少高端住宅项目中的隐藏成本，提升预算稳定性。

---

## Core Principle

预算失控，

往往不是因为开始的预算不够，

而是因为没有提前把交付复杂性考虑进去。

---

## Cost Structure

### 01 Material Cost

材料成本包括：

- 产品单价
- 材料数量
- 包装费用
- 标准运输
- 正常损耗
- 标准安装

特点：

- 公开透明
- 可提前估算
- 可以比较
- 相对稳定

### 02 Hidden Cost

隐藏成本包括：

- 返工
- 二次加工
- 补货
- 紧急运输
- 施工等待
- 工期延误
- 人工增加
- 责任争议
- 客户投诉
- 品牌信任损耗

特点：

- 难以预估
- 容易忽略
- 随项目变化
- 具有连锁影响
- 实际占比可能更高

## Six Critical Cost-Control Nodes

### 01 Finished Floor Level

施工前需要确认：

- 基层标高
- 地板厚度
- 胶黏剂厚度
- 石材厚度
- 瓷砖完成面
- 门槛高度
- 柜体完成面
- 楼梯完成面

### 02 Stair Detail

施工前需要确认：

- 踏步高度
- 踏步深度
- 楼梯鼻尺寸
- 立板厚度
- 结构基层
- 收边方式
- 灯带关系
- 起步与收尾节点

### 03 Transition & Trim

施工前需要确认：

- T型条
- 收口条
- Reducer
- End Cap
- 金属压条
- 无压条收口
- 不同材料高差
- 收口宽度

### 04 Expansion Gap

施工前需要确认：

- 伸缩缝位置
- 伸缩缝宽度
- 房间分区
- 门口位置
- 柜体周边
- 墙体周边
- 长距离铺装区域
- 地暖区域

### 05 Skirting

施工前需要确认：

- 踢脚线高度
- 踢脚线厚度
- 材质
- 颜色
- 安装方式
- 门套关系
- 柜体关系
- 阴阳角做法

### 06 Door Threshold

施工前需要确认：

- 门口材料交界
- 门扇开启高度
- 门套尺寸
- 地面高差
- 门槛石关系
- 收口型材
- 防水要求
- 视觉连续性

## Chain Cost Model

### Stage 01｜Node Unconfirmed

节点没有在设计阶段确定。

### Stage 02｜Site Decision

施工团队在现场临时判断。

### Stage 03｜Original Plan Invalid

原材料、尺寸或工艺无法直接使用。

### Stage 04｜Reprocessing

重新加工或修改材料。

### Stage 05｜Replenishment

增加补货和紧急运输。

### Stage 06｜Installation Change

施工顺序改变，安装难度增加。

### Stage 07｜Schedule Delay

其他专业等待，工期延长。

### Stage 08｜Budget Overrun

人工、运输、材料与管理成本同步增加。

## PDMS Budget Control Process

### Step 01｜Collect Project Information

收集：

- 设计图纸
- 材料清单
- 项目报价
- 施工条件
- 节点要求
- 工期计划

### Step 02｜Identify Critical Nodes

识别会影响预算的关键节点：

- 标高
- 收边
- 楼梯
- 踢脚线
- 门口
- 伸缩缝

### Step 03｜Complete Detailed Design

施工前完成：

- 节点图
- 尺寸确认
- 材料确认
- 工艺确认
- 责任确认

### Step 04｜Evaluate Chain Impact

每个节点都评估：

- 是否影响其他工种
- 是否影响材料加工
- 是否影响施工顺序
- 是否影响工期
- 是否增加成本

### Step 05｜Confirm Budget Boundary

明确：

- 标准费用
- 潜在风险费用
- 变更费用
- 责任归属
- 审批流程

### Step 06｜Control During Construction

通过：

- 现场巡检
- 节点验收
- 变更闭环
- 进度追踪
- 材料管理

减少隐藏成本。

### Step 07｜Final Review

复盘：

- 实际成本
- 预算偏差
- 返工原因
- 节点遗漏
- 改进措施

---

## Budget Control Formula

清晰设计  
+ 节点前置  
+ 材料协调  
+ 责任确认  
+ 过程管控  
+ 变更闭环  
= 稳定预算

提前规划，减少不确定性。

### 减少返工

节点提前确认，施工更顺畅。

### 降低风险

98个控制节点、120项验收标准。

### 提升效率

沟通更高效，决策更及时。

### 守住利润

减少隐性支出，避免项目利润被稀释。
