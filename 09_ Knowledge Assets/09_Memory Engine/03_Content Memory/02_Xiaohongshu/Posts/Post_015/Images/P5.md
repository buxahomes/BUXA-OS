# P5

## Page Role

PDMS Value Explanation

## Headline

PDMS  
为什么强调  
交付管理？

## Core Answer

因为真正需要控制的，

不是材料成本，

而是整个项目的隐藏成本。

## Five PDMS Values

### 可控预算

提前规划，减少不确定性。

预算更清晰，成本更可控。

### 减少返工

节点提前确认，施工更顺畅。

返工减少，成本自然下降。

### 降低风险

98个控制节点，

120项验收标准。

系统化管理，风险大幅降低。

### 提升效率

沟通更高效，决策更及时。

节省时间，也节省成本。

### 守住利润

减少隐性支出，

避免利润被稀释。

让项目真正实现预期价值。

## Bottom Statement

设计越完整，  
预算越稳定。

## Cost Comparison

### 材料成本

- 公开透明
- 可预估
- 可对比
- 比例较低

### 隐藏成本

- 难以预估
- 容易被忽略
- 随项目变化
- 占比更高

## Key Conclusion

控制隐藏成本，

才是控制预算的关键。
