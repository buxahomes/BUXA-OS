# P3

## Page Role

Chain Cost Explanation

## Headline

一个节点，  
可能带来的连锁成本

## Example

楼梯收口

## Scenario

如果施工当天才决定，

可能意味着：

### 重新加工

材料浪费，增加成本。

### 增加运输

紧急补货，费用更高。

### 增加安装

工序变化，安装难度增加。

### 增加人工

返工带来额外人工费用。

## Cost Chain

节点未定  
→ 成本增加  
→ 工期延长  
→ 预算超支

## Key Warning

一个节点，

可能影响整个施工节奏。

## Bottom Conclusion

提前确定每一个关键节点，

是控制预算的关键。

## Visual Description

右侧为木地板与混凝土楼梯或石材节点的剖面细节，展示金属收口构件、地板层及基层结构。

## Visual Purpose

用具体楼梯收口案例，将抽象的预算超支转化为可理解的成本链。
