## Visual Description

画面左侧为白色极简信息区，右侧为木地板施工节点的近景剖面。

地板一部分已完成铺设，另一部分基层及结构仍然外露，形成“完成状态”与“未解决节点”的视觉对比。

## Visual Metaphor

表面看起来只是木地板施工，但真正可能引发预算失控的是下方未被解决的结构、标高与收口问题。

## Visual Purpose

用“预算总会超”的直接问题抓住设计师与业主注意力，并将预算风险从材料价格转向交付节点。
