# P4

## Page Role

Cost and Risk Explanation

## Headline

施工前解决的问题，  
成本最低。

## Supporting Statement

发现问题的时间不同，

付出的成本和代价完全不同。

## Five Discovery Stages

### 设计阶段发现

成本最低，最容易解决。

### 施工开始前发现

成本较低，容易调整。

### 施工过程中发现

返工、延期、沟通成本增加。

### 安装完成后发现

返工成本高，影响验收。

### 入住以后发现

投诉、维修、品牌风险。

## Main Graphic

问题发现时间与成本关系曲线

横轴：

时间

纵轴：

成本 / 代价

随着问题发现时间推迟，成本呈明显上升趋势。

## Key Statement

成本最高，

代价最大。

## Bottom Conclusion

施工前的预防管理，

决定项目的最终交付质量。

## Visual Purpose

解释为什么交底必须前置，并将风险转化为直观的时间与成本关系。
