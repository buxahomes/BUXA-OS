# P2

## Page Role

Problem Introduction

## Headline

很多项目，  
不是施工不好。

而是施工开始之前，  
没有统一认知。

## Misalignment Statement

设计师以为图纸表达清楚了，

施工团队以为现场再沟通，

供应商以为施工知道要求。

结果：

每个人理解都不同。

## Left Section

### 没有统一交底

理解各不相同

#### 设计师

“我觉得这样就可以了。”

#### 施工方

“现场看情况再调整吧。”

#### 供应商

“按常规做法应该没问题。”

#### 业主

“我相信你们专业。”

## Result

施工中不断调整，

返工、延期、成本增加。

## Right Section

### 统一交底会议

认知一致，执行一致

#### 设计师

讲清设计意图

#### 施工方

明确施工做法

#### 供应商

确认材料节点

#### 业主

了解执行标准

## Key Conclusion

在施工前统一认知，

让项目从第一天就走在正确的轨道上。

## Bottom Keywords

- 统一标准
- 统一沟通
- 统一节点
- 统一验收

## Visual Purpose

通过“无交底”与“统一交底”对比，说明认知差异如何导致现场返工。
