## Visual Description

画面为高端住宅项目的专业交底会议。

主讲人员站在屏幕前讲解平面图与材料节点，其他项目人员围桌记录与讨论。

## Visual Purpose

将“施工前交底”上升为BUXA PDMS™统一标准、责任与验收体系的品牌方法论。
