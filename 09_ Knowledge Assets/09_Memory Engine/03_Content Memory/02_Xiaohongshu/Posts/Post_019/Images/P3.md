# P3

## Page Role

Inspection Checklist

## Headline

基层检查

到底检查什么？

## Supporting Statement

每一项检查，

都是为了确保地板安装后

长期稳定、美观、耐用。

## Item 01

### 含水率

检测基层含水率，

确保在安全范围内，

避免后期起拱、变形。

## Item 02

### 平整度

检测地面平整度，

确保铺设后地板

平整、舒适。

## Item 03

### 强度

检查基层强度，

确保承重和稳定性

符合安装要求。

## Item 04

### 空鼓裂缝

检查是否存在空鼓、

裂缝等问题，

避免后期隐患。

## Key Statement

安装前确认，

远比安装后返工更重要。

## Bottom Values

- 提前识别问题
- 数据真实记录
- 达标才可施工
- 专业标准执行
- 高质量交付

## Visual Description

页面以四个卡片展示含水率检测仪、水平尺、强度检测工具以及空鼓裂缝现场。

下方深色横条强调安装前确认的重要性。

## Visual Purpose

将基层验收具体化为四项核心检测内容，增强内容的可收藏和执行价值。
