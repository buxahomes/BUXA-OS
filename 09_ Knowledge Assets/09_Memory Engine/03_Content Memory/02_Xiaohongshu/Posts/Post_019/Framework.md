# Post_019 Framework

## Framework Name

PDMS Subfloor Inspection and Release Framework

## Chinese Name

PDMS基层验收与施工放行框架

## Framework Purpose

在木地板安装前，对基层含水率、平整度、强度及空鼓裂缝等条件进行系统检测、记录、判定和整改闭环，确保基层满足安装要求后再进入施工。

---

# Core Principle

先验收基层，

再开始铺装。

---

# Quality Gate

基层验收是木地板安装前的质量闸口。

只有满足以下条件，才允许进入安装：

- 检测项目完整
- 检测数据真实
- 检测结果达标
- 缺陷完成整改
- 复检结果合格
- 记录完成归档
- 责任人确认
- 施工条件满足

---

# Four-Dimension Subfloor Inspection Model

## Dimension 01｜Moisture Content

中文：

基层含水率

### Inspection Purpose

判断基层内部水分是否在安全范围内，降低木材吸湿、变形、起拱及胶粘失效风险。

### Inspection Content

- 水泥基层含水状态
- 不同房间含水率差异
- 临窗区域
- 卫生间相邻区域
- 地暖区域
- 新找平区域
- 施工用水影响区域
- 墙边和阴角区域

### Inspection Method

- 使用专业基层含水率检测设备
- 按区域多点检测
- 记录最高值、最低值和平均值
- 对异常区域进行复测
- 必要时增加环境温湿度记录

### Key Question

基层是否已经充分干燥，并满足当前安装系统要求？

### Risk if Missing

- 地板起拱
- 木材变形
- 板缝变化
- 胶粘失效
- 霉变风险
- 后期争议

---

## Dimension 02｜Flatness

中文：

基层平整度

### Inspection Purpose

判断基层高低差是否满足铺装要求，确保板面平整、脚感稳定并控制胶层厚度。

### Inspection Content

- 整体平整度
- 局部高点
- 局部低洼
- 墙边区域
- 门口区域
- 楼梯连接区域
- 材料交界区域
- 找平修补区域

### Inspection Method

- 使用靠尺
- 使用水平尺
- 使用激光水平仪
- 按网格或区域检测
- 标记超差位置
- 记录偏差值及范围

### Key Question

基层高低差是否会影响铺装、收口或使用脚感？

### Risk if Missing

- 地板异响
- 板面悬空
- 拼缝不平
- 脚感不稳定
- 胶层过厚或不均
- 局部变形

---

## Dimension 03｜Strength

中文：

基层强度

### Inspection Purpose

确认基层具备承载地板和粘结系统的能力，不发生起砂、松散或脱层。

### Inspection Content

- 表层硬度
- 起砂情况
- 松散情况
- 找平层结合情况
- 局部修补区域
- 裂缝边缘强度
- 胶粘施工承载能力
- 基层整体稳定性

### Inspection Method

- 表面观察
- 划痕检查
- 敲击检查
- 专业强度检测
- 必要时进行拉拔或其他专项测试
- 对疑似松散区域重点检测

### Key Question

基层是否能够可靠承受安装、粘结及长期使用荷载？

### Risk if Missing

- 胶层脱落
- 基层粉化
- 地板局部松动
- 空鼓脱层
- 使用异响
- 大面积返工

---

## Dimension 04｜Hollowing and Cracks

中文：

空鼓、裂缝与基层缺陷

### Inspection Purpose

识别可能导致后期位移、脱层、异响和结构性问题的基层缺陷。

### Inspection Content

- 空鼓
- 明显裂缝
- 贯穿裂缝
- 收缩裂缝
- 修补裂缝
- 基层脱层
- 起砂
- 污染
- 凸起
- 凹陷
- 残留物
- 表面松散

### Inspection Method

- 敲击检测
- 视觉检查
- 标记缺陷范围
- 记录裂缝方向和宽度
- 拍照留档
- 对疑似结构性裂缝提交专项评估

### Key Question

基层是否存在可能在铺装后继续发展的隐患？

### Risk if Missing

- 局部空鼓
- 地板异响
- 地板松动
- 裂缝传递
- 胶层失效
- 重复维修

# Extended Inspection Items

除四个核心项目外，基层验收还应关注：

## 01 Cleanliness

- 灰尘
- 油污
- 石膏残留
- 涂料残留
- 胶渍
- 松散颗粒
- 建筑垃圾

## 02 Temperature and Humidity

- 室内温度
- 室内相对湿度
- 地暖运行状态
- 通风状态
- 门窗封闭条件

## 03 Finished Floor Level

- 基层标高
- 找平层厚度
- 地板系统厚度
- 门洞高度
- 楼梯首末级高度
- 其他材料完成面

## 04 Environmental Conditions

- 是否存在渗水
- 是否存在漏水
- 外门窗是否完成
- 湿作业是否结束
- 现场是否适合存放木地板
- 其他工种是否会破坏基层

# PDMS Subfloor Inspection Process

## Step 01｜Pre-Inspection Preparation

收集：

- 最新施工图
- 地面铺装图
- 完成面标高图
- 基层施工资料
- 找平材料信息
- 地暖系统信息
- 安装材料及胶水要求
- 项目施工计划
- 已知问题记录

## Step 02｜Divide Inspection Zones

按空间划分检测区域：

- 客厅
- 餐厅
- 卧室
- 走廊
- 楼梯
- 门口
- 临窗区域
- 地暖区域
- 异形空间
- 材料交界区域

每个区域建立唯一编号。

## Step 03｜Moisture Test

对不同区域进行多点含水率检测。

### Record Fields

- 区域编号
- 测点编号
- 测试位置
- 检测数值
- 环境温度
- 环境湿度
- 设备编号
- 检测人员
- 检测时间
- 是否达标

## Step 04｜Flatness Test

使用靠尺、水平尺或激光设备检测基层高低差。

### Record Fields

- 检测区域
- 检测方向
- 偏差值
- 超差范围
- 处理建议
- 复检结果

## Step 05｜Strength Test

检查基层强度、起砂、松散和脱层情况。

### Classification

#### Grade A

基层坚实，无明显缺陷。

#### Grade B

局部轻微问题，可修补后施工。

#### Grade C

存在明显松散或强度不足，必须整改。

#### Grade D

基层整体不满足要求，不得施工。

## Step 06｜Defect Mapping

将空鼓、裂缝、起砂和污染区域标记在平面图中。

### Defect Record

- 问题编号
- 问题类型
- 具体位置
- 影响范围
- 风险等级
- 处理建议
- 责任单位
- 整改期限
- 复检状态

## Step 07｜Data Recording

形成：

- 检测记录表
- 测点数据
- 现场照片
- 平面标记图
- 问题清单
- 整改建议
- 责任记录
- 检测结论

## Step 08｜Assessment

检测结论分为：

### Pass

符合安装要求，可以进入下一步。

### Conditional Pass

局部问题整改完成并复检后，可以施工。

### Fail

基层条件不满足，不得开始铺装。

## Step 09｜Rectification

不达标时：

- 提出整改建议
- 明确整改责任人
- 明确整改完成时间
- 暂停地板安装
- 整改完成后重新检测
- 复检合格后释放施工

## Step 10｜Installation Release

只有满足以下条件，才能安排施工：

- 含水率达标
- 平整度达标
- 强度达标
- 空鼓裂缝完成处理
- 清洁度符合要求
- 环境条件满足
- 检测数据归档
- 验收报告签署
- 施工计划确认

---

# Four-Step Closed Loop

## 01 基层检测

检测含水率、平整度、强度、空鼓裂缝等项目。

## 02 数据记录

逐项记录数据，保存照片和现场情况。

## 03 达标确认

将检测结果与项目安装标准进行对照。

## 04 安排施工

只有验收合格后，才能正式进入铺装阶段。

# Subfloor Decision Matrix

## Condition A｜Fully Compliant

处理方式：

正常安排施工。

## Condition B｜Minor Local Defects

处理方式：

局部整改，复检合格后施工。

## Condition C｜Moisture Above Limit

处理方式：

延长干燥时间、改善通风或进行专项处理，暂停施工。

## Condition D｜Flatness Out of Tolerance

处理方式：

重新找平或局部修补，完成后复测。

## Condition E｜Insufficient Strength

处理方式：

加固、铲除重做或采用经确认的基层处理方案。

## Condition F｜Hollowing or Major Cracks

处理方式：

查明原因，完成专业修复后重新验收。

# Risk Prevention Logic

先检查基层  
→ 提前发现问题  
→ 在地板铺装前整改  
→ 避免缺陷被覆盖  
→ 降低后期起拱、异响和空鼓风险  
→ 减少维修与返工  
→ 提升长期使用品质

# Responsibility Model

## BUXA Project Engineer

- 组织基层检查
- 使用专业设备检测
- 记录检测数据
- 标记问题区域
- 出具验收意见
- 跟踪整改和复检
- 确认是否放行施工

## Main Contractor / Subfloor Contractor

- 提供合格基层
- 对不合格问题进行整改
- 配合复检
- 对基层施工质量负责

## Flooring Installation Team

- 确认基层验收完成
- 按批准条件施工
- 不得覆盖未处理问题
- 记录安装前现场状态

## Interior Designer

- 了解基层状况对设计落地的影响
- 协调相关界面和完成面
- 参与重大问题决策

## Homeowner / Project Representative

- 了解不达标施工的风险
- 支持整改和施工节点调整
- 对关键验收结果进行确认

# Framework Conclusion

基层验收不是在检查一块水泥地。

它是在确认整个木地板系统，

是否具备长期稳定运行的基础。
