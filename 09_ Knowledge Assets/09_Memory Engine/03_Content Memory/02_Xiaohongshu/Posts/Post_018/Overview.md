# Post_018 Overview

## One-Sentence Summary

材料正式下单前进行现场复核，是为了让图纸、现场、材料和施工条件重新匹配，避免材料生产以后才发现尺寸、节点和安装条件发生变化。

## Core Question

既然施工图已经确认，为什么材料下单前还必须到现场复核？

## Core Answer

因为施工图记录的是设计确认时的理想状态，而项目现场处于持续变化之中；只有通过现场复核，才能确认材料生产所依据的数据仍然真实、完整和可执行。

## Central Thesis

图纸确认解决“设计是否成立”，现场复核解决“材料能否准确落地”。

## Main Message

材料下单不是单纯的采购动作，而是从设计方案进入生产和施工执行的重要控制节点。

## Supporting Message

完成面标高、基层条件、门洞尺寸、楼梯节点、收口位置、运输路线、堆放空间和安装顺序，只要有一项发生变化，都可能导致补货、返工、延期或效果偏差。

## Key Takeaway

图纸是一套理想状态，而现场永远是真实状态。

### Before Reading

施工图和材料规格确认后即可下单生产，现场尺寸问题可以在安装阶段调整。

### After Reading

材料下单前必须重新核验现场，因为一旦进入生产，任何误差都会转化为材料、时间和施工成本。
