# Post_018 Framework

## Framework Name

PDMS Pre-Order Site Verification Framework

## Chinese Name

材料下单前现场复核框架

## Framework Purpose

在材料正式生产前，对施工现场、图纸、尺寸、节点、运输及安装条件进行系统核验，确保材料能够一次到位并顺利安装。

---

# Core Principle

没有完成现场复核，

就不应该进入正式下单。

---

# Site Verification Is Not Measurement Only

现场复核不等于重新测量几个尺寸。

完整的现场复核应确认：

1. 图纸与现场是否一致
2. 尺寸是否可以用于生产
3. 完成面关系是否成立
4. 基层是否满足施工要求
5. 节点是否可以实际安装
6. 材料数量是否准确
7. 材料是否能够运输进场
8. 现场是否具备堆放条件
9. 安装顺序是否已经明确
10. 生产和安装时间是否匹配

---

# Seven-Layer Site Verification Model

## Layer 01｜Dimensional Verification

中文：

现场尺寸复核

### Confirm

- 房间长度与宽度
- 墙体位置
- 门洞宽度与高度
- 柜体尺寸
- 柱体尺寸
- 楼梯尺寸
- 转角位置
- 材料交界线
- 不规则区域尺寸
- 实际铺装面积

### Key Question

生产尺寸和材料数量是否建立在真实现场数据上？

### Risk if Missing

- 地板数量错误
- 收口尺寸不匹配
- 材料无法安装
- 补货或剩余过多
- 生产计划失准

---

## Layer 02｜Finished Floor Level Verification

中文：

完成面标高复核

### Confirm

- 各空间完成面标高
- 地板结构总厚度
- 石材或瓷砖完成面
- 地暖结构厚度
- 找平层厚度
- 门槛高度
- 楼梯首末级高度
- 柜体底部关系
- 室内外高差

### Key Question

不同材料和不同空间的完成面是否可以顺利衔接？

### Risk if Missing

- 高低差错误
- 门无法开启
- 楼梯首末级不一致
- 收口型材无法使用
- 现场重新找平

---

## Layer 03｜Subfloor Verification

中文：

基层条件复核

### Confirm

- 平整度
- 强度
- 含水率
- 清洁度
- 裂缝情况
- 空鼓情况
- 高低差
- 地暖运行状态
- 防潮层条件
- 找平完成状态

### Key Question

现场是否已经具备材料安装条件？

### Risk if Missing

- 地板空响
- 起拱
- 变形
- 胶粘失效
- 工期等待
- 后期质量隐患

---

## Layer 04｜Interface Verification

中文：

节点与收口复核

### Confirm

- 地板与石材收口
- 地板与瓷砖收口
- 地板与楼梯连接
- 地板与门套关系
- 地板与踢脚线关系
- 地板与柜体关系
- 地板与隐藏门关系
- 伸缩缝位置
- 转角及异形节点
- 室内外门口节点

### Key Question

所有材料接口是否已经明确，并可以实际施工？

### Risk if Missing

- 收口位置冲突
- 临时更换型材
- 材料切割错误
- 视觉效果偏差
- 现场反复协调

---

## Layer 05｜Material Quantity Verification

中文：

材料数量复核

### Confirm

- 实际铺装面积
- 损耗率
- 异形区域损耗
- 楼梯材料数量
- 收口条数量
- 踢脚线数量
- 备用材料数量
- 批次一致性要求
- 包装单位
- 补货周期

### Key Question

材料数量是否足够、合理且可追溯？

### Risk if Missing

- 数量不足
- 批次色差
- 临时补货
- 数量过剩
- 成本浪费
- 工期等待

---

## Layer 06｜Logistics Verification

中文：

运输与堆放条件复核

### Confirm

- 车辆能否到达
- 卸货区域
- 电梯尺寸
- 楼梯通道
- 门洞净宽
- 转弯空间
- 材料长度限制
- 搬运人员
- 临时堆放区域
- 防潮、防晒及成品保护条件

### Key Question

材料生产完成后，是否能够顺利进入现场并安全存放？

### Risk if Missing

- 材料无法进场
- 二次搬运
- 包装拆解
- 材料损坏
- 运输费用增加
- 施工等待

---

## Layer 07｜Installation Sequence Verification

中文：

安装顺序复核

### Confirm

- 地面找平完成时间
- 地暖检测时间
- 地板进场时间
- 材料养生时间
- 柜体施工时间
- 门套安装时间
- 楼梯施工时间
- 踢脚线安装时间
- 灯带施工时间
- 成品保护时间
- 最终验收时间

### Key Question

材料生产与项目实际施工节奏是否匹配？

### Risk if Missing

- 材料提前到场
- 材料无法存放
- 工序交叉
- 成品损坏
- 施工延期
- 重复进场

# PDMS Pre-Order Verification Process

## Step 01｜Prepare

现场复核前收集：

- 最新施工图
- 地面铺装图
- 完成面标高图
- 楼梯节点图
- 门表
- 柜体图
- 踢脚线方案
- 材料规格
- 现场施工计划
- 既有变更记录

## Step 02｜Verify Drawing Version

确认现场使用的图纸与下单依据是同一个版本。

### Required Fields

- 图纸编号
- 版本号
- 修改日期
- 确认人
- 变更内容
- 当前状态

## Step 03｜Measure

对关键尺寸进行现场测量和记录。

### Measurement Records

- 测量位置
- 图纸尺寸
- 现场尺寸
- 偏差值
- 是否影响生产
- 是否需要调整
- 照片证据

## Step 04｜Inspect Conditions

检查：

- 完成面
- 基层
- 门洞
- 楼梯
- 节点
- 材料交界
- 运输路线
- 堆放条件
- 施工进度

## Step 05｜Identify Variations

将现场与图纸之间的差异分类：

### Category A

可接受偏差，不影响生产。

### Category B

需调整图纸或材料数量。

### Category C

需深化节点后才能下单。

### Category D

现场条件不满足，暂缓下单。

## Step 06｜Resolve

针对每一项偏差确认：

- 问题描述
- 影响范围
- 调整方案
- 责任人
- 完成日期
- 验证方式
- 是否影响工期
- 是否影响成本

## Step 07｜Approve

形成正式下单前确认记录。

### Required Sign-Off

- 设计师
- 项目经理
- BUXA项目工程师
- 施工负责人
- 材料负责人
- 必要时业主代表

## Step 08｜Release Order

只有满足以下条件后，才正式释放生产订单：

- 图纸版本锁定
- 现场尺寸确认
- 完成面确认
- 节点确认
- 数量确认
- 运输条件确认
- 安装计划确认
- 风险闭环完成
- 责任人签字确认

---

# Seven Mandatory Verification Items

## 01 完成面标高

检查各空间完成面高度，确认地板、石材、门、楼梯和柜体关系。

## 02 基层条件

检查平整度、强度、含水率和清洁度。

## 03 门洞尺寸

复核门洞宽度、高度及门套安装空间。

## 04 楼梯节点

复核踏步尺寸、首末级高度、收口和安装结构。

## 05 收口方案

确认不同材料交界位置和收口型材。

## 06 运输条件

确认材料尺寸能够通过电梯、楼梯及门洞，并具备堆放空间。

## 07 安装顺序

确认材料进场、安装和其他工种之间的先后关系。

---

# Site Verification Decision Matrix

## Condition A｜All Confirmed

结果：

可以下单。

## Condition B｜Minor Deviation

结果：

修订数量或图纸后下单。

## Condition C｜Node Unconfirmed

结果：

完成深化设计后再下单。

## Condition D｜Site Not Ready

结果：

暂缓生产，待现场条件满足后复核。

## Condition E｜Major Design Change

结果：

重新确认设计、尺寸、材料和预算。

---

# Order Release Formula

最新图纸  
+ 现场实测数据  
+ 完成面确认  
+ 基层条件确认  
+ 节点深化  
+ 精准材料数量  
+ 运输条件确认  
+ 安装顺序确认  
+ 风险闭环  
= 正式下单授权

---

# Risk Prevention Formula

现场复核  
→ 提前发现偏差  
→ 调整图纸与数量  
→ 材料精准生产  
→ 施工顺利推进  
→ 减少补货与返工  
→ 高质量交付

---

# Framework Conclusion

真正的现场复核，

不是为了证明图纸有问题。

而是为了确认：

材料在生产以后，

可以准确进入现场，

并一次完成交付。
