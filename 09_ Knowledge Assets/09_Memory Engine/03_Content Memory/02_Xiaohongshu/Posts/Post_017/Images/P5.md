# P5

## Page Role

Management Value

## Headline

PDMS 关注的

不是压缩预算，

而是控制预算变化。

## Supporting Statement

通过系统化管理，

让每一分钱花在正确的位置。

## Management Value 01

### 节点前置确认

关键节点提前确认，

减少后期变更。

## Management Value 02

### 施工流程控制

标准化施工流程，

保障执行效率。

## Management Value 03

### 风险提前发现

提前识别风险，

避免问题扩大。

## Management Value 04

### 降低返工概率

减少返工和重复施工，

控制隐性成本。

## Management Value 05

### 多方协同管理

强化跨专业协同，

减少沟通损耗。

## Management Value 06

### 预算稳定可控

让预算更可控，

让项目更安心。

## Bottom Quote

真正的成本控制，

来自于系统管理，

而不是事后补救。

## Process Summary

提前计划  
→ 过程控制  
→ 降低风险  
→ 成本可控

## Visual Description

上方为正在施工中的高端住宅空间。

下方使用六个管理价值模块，说明PDMS如何通过施工前和施工中的系统控制降低预算波动。

## Visual Purpose

把“成本控制”从压价逻辑转变为风险和变化管理逻辑。
