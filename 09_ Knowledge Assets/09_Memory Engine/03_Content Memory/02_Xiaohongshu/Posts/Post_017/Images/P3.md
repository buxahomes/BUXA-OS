# P3

## Page Role

Cost Escalation Explanation

## Headline

预算为什么

越来越高？

## Supporting Statement

每一次变化，

都会放大整个项目成本。

## Project Stages

### 设计阶段

DESIGN STAGE

对应风险：

设计调整

方案变更，增加工作量。

### 施工阶段

CONSTRUCTION STAGE

对应风险：

材料返工

不符合要求，拆除重做。

### 安装阶段

INSTALLATION STAGE

对应风险：

工期延误

等待协调，延长工期。

### 交付阶段

COMPLETION STAGE

对应风险：

重复施工

多次进场，成本叠加。

## Key Conclusion

问题越早发现，成本越低；

问题越晚发现，成本越高。

## Visual Description

右侧为带尺寸标注的住宅平面图，并用红色虚线圈出潜在冲突位置。

左侧展示项目从设计到交付的四个阶段及相应成本风险。

底部使用上升柱状图表达发现问题越晚，成本增长越快。

## Visual Purpose

建立“风险发现时间与解决成本”之间的关系。
