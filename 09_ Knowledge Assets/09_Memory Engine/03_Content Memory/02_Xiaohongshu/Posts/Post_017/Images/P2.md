## Visual Description

左侧为暖白色文字区。

右上为设计师在图纸和材料样板前审核施工图。

右下三张小图分别呈现毛坯现场、地板安装和墙体放线等施工状态。

底部黑色栏用四个图标总结未确认图纸可能带来的后果。

## Visual Purpose

将施工风险的起点从“现场”前移到“图纸确认阶段”，建立本篇核心问题。
