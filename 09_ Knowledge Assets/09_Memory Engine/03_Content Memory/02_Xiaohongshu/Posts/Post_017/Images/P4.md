# P4

## Page Role

PDMS Prevention Process

## Headline

很多成本，

其实可以提前避免。

## Supporting Statement

通过 PDMS 在施工前建立系统化的节点控制，

把风险消灭在萌芽阶段。

## Step 01

### 设计复核

- 图纸完整性检查
- 节点细节确认
- 标高与尺寸核验
- 材料方案确认

## Step 02

### 材料规划

- 材料规格确认
- 样品效果确认
- 供应周期管理
- 进场计划制定

## Step 03

### 现场协调

- 各专业图纸会审
- 现场条件复核
- 安装顺序规划
- 风险点提前识别

## Step 04

### 安装控制

- 关键节点验收
- 安装工艺控制
- 问题闭环管理
- 过程质量跟踪

## Key Conclusion

问题越早发现，

成本越低。

## Visual Description

页面以四个竖向流程卡片展示PDMS从设计审核到安装控制的完整流程。

底部用折线图表达成本随问题发现时间而上升。

## Visual Purpose

说明PDMS不是施工后的补救工具，而是施工前的风险预防机制。
