# Post_017 Overview

## One-Sentence Summary

高端住宅项目的风险往往不是施工后突然出现，而是在施工图和关键节点未被充分确认时就已经形成。

## Core Question

为什么设计师必须在施工开始前确认地面系统深化图和关键节点图？

## Core Answer

因为施工图不仅表达设计意图，还承担尺寸协调、材料衔接、工序规划、风险识别和施工执行依据等功能。

## Central Thesis

施工图确认不是设计流程的结束，而是高质量交付真正开始的管理节点。

## Main Message

设计师需要确认的，不只是空间“看起来怎么样”，还包括空间“具体如何实现”。

## Supporting Message

完成面标高、材料收口、楼梯尺寸、门套关系、踢脚线关系和材料进场顺序，只要有一项未确认，都可能在施工阶段形成连锁风险。

## Key Takeaway

问题越早被发现，解决成本越低；问题越晚被发现，代价越高。

## Reader Transformation

### Before Reading

施工图主要用于表达设计和指导施工，现场遇到问题再协调即可。

### After Reading

施工图必须在开工前完成跨专业核验，因为很多返工、冲突和延期，在图纸阶段就已经可以被发现。
