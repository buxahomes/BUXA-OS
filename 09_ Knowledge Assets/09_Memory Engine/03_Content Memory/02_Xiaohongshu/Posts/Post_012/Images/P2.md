# P2

## Page Role

Misconception Correction

## Headline Left

很多人认为：

只要买贵一点，  
效果自然会更好。

## Headline Right

实际上，

真正决定最终效果的，

往往不是地板，

而是施工过程。

## Comparison Title

同一种木地板

## Left Side

### 普通施工

- 拼缝不一致，高低差明显
- 地板反射杂乱，整体感弱
- 与空间融合度低，质感一般

## Right Side

### 精细管理施工

- 拼缝整齐，衔接自然
- 地板反射柔和，光感高级
- 与空间风格统一，整体质感提升

## Visual Description

下半部分展示两个使用相同木地板的客厅空间。

左侧整体平淡、拼缝及光线效果较弱。

右侧排版统一、空间关系协调，整体视觉更加高级。

## Visual Purpose

打破“贵材料自然等于好效果”的认知，将注意力转移到施工与管理。
