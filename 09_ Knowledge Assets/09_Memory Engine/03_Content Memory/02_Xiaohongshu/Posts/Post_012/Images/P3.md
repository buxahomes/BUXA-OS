# P3

## Page Role

Key Detail Checklist

## Headline

真正影响最终效果的，  
通常是这些细节

## Seven Key Details

### 01 基层是否合格

平整度、强度、含水率是否符合要求。

### 02 含水率是否检测

地面含水率、木材含水率是否在标准范围内。

### 03 铺装方向是否提前规划

空间比例、光线走向与地板方向是否协调。

### 04 起铺点是否统一

起铺点位置是否合理，视觉是否对称一致。

### 05 收边尺寸是否计算

与墙面、柜体、门套的收边尺寸是否精准。

### 06 预留伸缩缝是否正确

伸缩缝位置、宽度是否符合规范要求。

### 07 楼梯节点是否提前深化

踏步收边、压条、转角处理是否提前确认。

## Supporting Statement

这些，设计图纸通常不会告诉你。

## Visual Purpose

将最终效果差异拆解成七个可检查、可管理的具体项目节点。
