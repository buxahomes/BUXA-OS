# AI Workflow

Before generating any new Xiaohongshu content:

1. Read this Index.
2. Locate similar historical posts.
3. Read only the related post files.
4. Avoid duplicate topics.
5. Preserve BUXA positioning.
6. Expand the existing knowledge graph instead of repeating previous ideas.

AI should never generate Xiaohongshu content without consulting this Index first.

5. Update Topic Coverage.

# Long-Term Vision

The Xiaohongshu Content Memory is not a collection of social posts.

It is the publishing memory of BUXA.

Its purpose is to help AI understand:

- what has been published;
- what has performed well;
- what topics have become authoritative;
- what content gaps remain;
- how the BUXA brand has evolved over time.

The long-term objective is to allow AI to plan future Xiaohongshu content from accumulated business memory rather than isolated prompts.
